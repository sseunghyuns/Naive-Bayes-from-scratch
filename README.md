# Naive-Bayes-from-scratch
Filtering spam mails using Naive Bayes algorithm.  
All codes were implemented without using external libraries.  

---

## Run

```
$ python main.py
```

---

## Result

Total train spam mails: 100  
Total train ham mails : 100  

Accuracy : 90%
